模板模式很好的说明了,抽象类是类抽象

建一个哺乳动物抽象类,
```java
package com.ydlclass.authdemo.mdsjms;  
  
public abstract class Mammal {  
    public abstract void eat();  
  
    public abstract void move();  
  
    public final void live() {  
        eat();  
        move();  
    }  
}
```

建一个人的实体类
```java
package com.ydlclass.authdemo.mdsjms;  
  
public class Human extends Mammal{  
    @Override  
    public void eat() {  
        System.out.println("人类吃吃吃");  
    }  
  
    @Override  
    public void move() {  
        System.out.println("人类在开车");  
    }  
}
```

建一个鲸鱼的实体类
```java
package com.ydlclass.authdemo.mdsjms;  
  
public class Whale  extends Mammal{  
    @Override  
    public void eat() {  
        System.out.println("鲸鱼吃吃吃");  
    }  
  
    @Override  
    public void move() {  
        System.out.println("鲸鱼游游游");  
    }  
}
```

测试
```java
package com.ydlclass.authdemo.mdsjms;  
  
public class Test {  
    public static void main(String[] args) {  
        Mammal whale = new Whale();  
        Mammal human = new Human();  
        whale.live();  
        human.live();  
    }  
}
```
